Reemplazar HTML entities por caracteres con tilde usando grep y sed

La versión anterior de este blog la cree usando un editor html (tinymce), el problema es que todos los caracteres especiales fueron reemplazados por html entities, con los siguientes comandos voy a cambiarlos.

grep -rl "&aacute;" . | xargs sed -i 's/&aacute;/á/g'
grep -rl "&eacute;" . | xargs sed -i 's/&eacute;/é/g'
grep -rl "&iacute;" . | xargs sed -i 's/&iacute;/í/g'
grep -rl "&oacute;" . | xargs sed -i 's/&oacute;/ó/g'
grep -rl "&uacute;" . | xargs sed -i 's/&uacute;/ú/g'
